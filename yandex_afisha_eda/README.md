# Анализ данных Яндекс Афиши (лето–осень 2024)

## Цель

Провести исследовательский анализ данных сервиса Яндекс Афиша, выявить сезонные изменения пользовательской активности осенью 2024 года и проверить гипотезы о различиях поведения пользователей мобильных и стационарных устройств.

## Данные

Использованы данные:

* заказы билетов;
* информация о событиях;
* курсы валют для приведения выручки к единой валюте.

## Основные результаты

* осенью наблюдается рост количества заказов и активных пользователей;
* средняя цена билета в целом стабильна, при этом по типам мероприятий выявлены различия;
* пользователи мобильных устройств совершают больше заказов, различие статистически значимо;
* значимых различий во времени между заказами для мобильных и десктопных устройств не обнаружено.

## Инструменты

Python, pandas, numpy, matplotlib, seaborn, scipy
